2.元组(tuple)
^^^^^^^^^^^^^^^^^^^

a.元组定义
-----------------
元组是一系列按特定排列的元素组成的序列，在Python中用()来表示

**注：Python中对于用逗号分开的量，即使不使用小括号也默认为元组**

如：1, 3, 5, 7 = (1, 3, 5, 7)

----

b.与列表的本质区别
---------------------
**元组为不可变对象，列表为可变对象**

`即元组也可以像列表一样切片，但不能像列表一样更改`

----

c.序列的分类
--------------------
①按照是否可变：
 | 可变：list，set，dict
 
   不可变：str，tuple


②按照是否有序：
 | 有序：list，tuple，str

   无序：set，dict
